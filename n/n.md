# N-Aliases — Алиасы для Node.js разработки

Набор bash-алиасов для быстрой работы с Node.js проектами через [ni](https://github.com/antfu-collective/ni).

## Установка

```bash
brew install ni
```

## Алиасы

### Управление зависимостями

| Алиас   | Команда                                              | Описание                                      |
|---------|------------------------------------------------------|-----------------------------------------------|
| `nu`    | `nup`                                                | Обновление зависимостей                       |
| `nrm`   | `rm -f {lock-файлы} && ni`                           | Удалить lock-файлы и переустановить           |
| `nrmf`  | `rm -f {lock-файлы} && rm -rf node_modules && ni`    | Полная переустановка (с удалением node_modules) |

### Запуск проекта

| Алиас | Команда         | Описание                                           |
|-------|-----------------|----------------------------------------------------|
| `n`   | `nr start`      | Запуск проекта                                     |
| `ns`  | `nr start:dev`  | Умный запуск dev-режима (start:dev, start:ts или storybook) |
| `nd`  | `nr dev`        | Режим разработки                                   |
| `nds` | `nr dev:server` | Dev-сервер                                         |
| `ndc` | `nr dev:client` | Dev-клиент                                         |

### Сборка

| Алиас | Команда             | Описание               |
|-------|---------------------|------------------------|
| `nb`  | `nr build`          | Сборка проекта         |
| `nbb` | `nr build --prod`   | Продакшн сборка        |
| `nbt` | `nr build && test`  | Сборка + тесты (prod)  |

### Тестирование

| Алиас | Команда                | Описание                                |
|-------|------------------------|-----------------------------------------|
| `nt`  | `nr test`              | Запуск тестов                           |
| `ntw` | `nr test:watch`        | Тесты в watch-режиме                    |
| `ntd` | `nr test:demo/dev`     | Тесты в demo/dev режиме                 |
| `ntl` | `nr test:lint`         | Линтинг                                 |

### Релиз

| Алиас | Команда            | Описание                        |
|-------|--------------------|---------------------------------|
| `nf`  | `nr fix`           | Автофикс                        |
| `np`  | `nr release`       | Релиз (с подтверждением)        |
| `npp` | `nr release --yes` | Релиз (без подтверждения)       |

## Умные алиасы

Некоторые алиасы автоматически определяют доступные скрипты в `package.json`:

- **`ns`** — ищет `start:dev` → `start:ts` → `storybook`
- **`ntw`** — ищет `start:watch` → `test:watch` → `test --watch`
- **`ntd`** — ищет `test:demo` → `test:dev` → `test --dev`
- **`ntl`** — ищет `test:lint` → `test:eslint` → `test --lint`

## Логирование

Все команды выводятся в консоль перед выполнением благодаря функции `log()`:

```
> nr build
```
