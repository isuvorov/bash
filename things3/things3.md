# Things3 Aliases

Работа с задачами в Things3 через командную строку. Добавление задач через Mail.app и просмотр списков.

## Функции

| Функция      | Описание                                    |
| ------------ | ------------------------------------------- |
| `todoadd`    | Добавить задачу в Things3 через Mail.app    |
| `todolist`   | Показать список задач из указанного списка  |

## Использование

### Добавление задачи

```bash
# Простая задача
todoadd купить хлеб

# Задача с темой
todoadd -s "Срочно" оплатить налоги

# Многострочная задача из stdin
todoadd -s "План" <<EOF
1. Сделать бэкап
2. Запустить деплой
3. Проверить логи
EOF

# Из вывода команды
ls -la | todoadd -s "Список файлов"
```

### Просмотр списка задач

```bash
# Список задач из Inbox (по умолчанию)
todolist

# Список задач из конкретного списка
todolist "Today"
todolist "Upcoming"
```

## Особенности

- `todoadd` использует Mail.app для отправки задач в Things3
- Требуется настройка переменной окружения `SENDTODO_EMAIL` с адресом для приёма задач
- Если тема не указана, используется "TODO" по умолчанию
- Поддерживает многострочный ввод через stdin
- `todolist` показывает пустое сообщение, если список пуст

## Настройка

Убедитесь, что переменная окружения `SENDTODO_EMAIL` настроена:

```bash
export SENDTODO_EMAIL="your-things3-email@example.com"
```
